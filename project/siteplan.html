From Local Fundamentals to FAANG Elite

Website Subject

Title: GlobalDev Roadmap: From Local Fundamentals to FAANG Elite.

The website is a professional resource hub designed to guide aspiring software engineers through a global career path. It features:

  • A Dynamic Resource Gallery: JavaScript arrays and objects display high-quality, free certifications from institutions like Harvard (CS50), Google (DevOps), and IBM (Data Science).
  • Filtering Logic: Interactive buttons allow users to filter resources by field (Computer Engineering, Web Development, AI), difficulty level, or region (Local South Africa vs. International Remote).
  • The Lean Developer Guide: Content focused on code optimization and fundamental principles for developers working in environments with high data costs or hardware constraints.
  • Save for Later: A localStorage feature that allows users to build and persist a personal learning plan across sessions.
  • Community Interaction: A standards-compliant HTML form for users to suggest new tools or community forums.

Reason for This Subject

As a Software Development student at BYU-Idaho living in South Africa, I have observed a gap between learning basic syntax and understanding the engineering rigor required by international corporate giants. I chose this subject to centralize high-weight, zero-cost resources that emphasize system efficiency and computer engineering fundamentals. This platform aims to empower students to build globally competitive careers while respecting the unique infrastructure challenges found in local markets.

Name: GlobalDev Roadmap

Purpose: To empower South African developers with high-weight international resources while optimizing for local infrastructure challenges through “Lean Development” principles.

Color Schema & Typography

Our design follows the 60-30-10 rule for visual balance:

  • 60% Dominant: #F8F9FA (Backgrounds)
  • 30% Secondary: #1B263B (Structure/Headers)
  • 10% Accent: #FF4D00 (Interactions/CTA)

Fonts: Inter (Headings) provides a modern tech feel, while Roboto (Body) ensures maximum readability across devices.
